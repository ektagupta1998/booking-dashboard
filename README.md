# Booking Dashboard Application

A full-stack Booking Dashboard application built as an assignment project.
This application allows users to create, view, update, and manage bookings
through a clean admin dashboard interface.

## Features

- Create new bookings using a side modal
- View all bookings in a responsive table
- Update booking status (Pending / Confirmed / Cancelled)
- Edit bookings using the same reusable modal
- Delete bookings with confirmation
- Frontend and backend validation
- Mobile responsive UI

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Fetch API

### Backend
- Node.js
- Fastify
- MongoDB
- Mongoose
- Swagger (API documentation)
